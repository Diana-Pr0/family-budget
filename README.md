# Family Budget Lite

Небольшой веб-проект для учета семейного бюджета: с приветственным лендингом, базовой аналитикой и локальным хранением данных.

## Что умеет

- добавлять доходы и расходы по категориям;
- показывать общий доход, расход, баланс и процент расходов;
- фильтровать историю операций;
- отображать статистику расходов по категориям;
- распознавать текст чека (OCR) и считывать QR-код с фото;
- задавать финансовую цель и видеть прогресс;
- экспортировать и импортировать данные в JSON.

## Стек

- чистый HTML/CSS/JavaScript (без сборки и серверной части);
- данные хранятся в `localStorage` браузера.

## Быстрый старт

1. Откройте файл `index.html` в браузере.
2. Начните добавлять операции.

Либо можно запустить локальный сервер:

```bash
python3 -m http.server 8000
```

После запуска откройте [http://localhost:8000](http://localhost:8000).

## Структура проекта

- `index.html` — главная страница приложения;
- `Семейный бюджет.html` — исходная версия страницы;
- `.gitignore` — базовые исключения для репозитория.

## Публикация

Проект подходит для GitHub Pages / Netlify / Vercel как статический сайт.
